## External API Integration

### Overpass API (OpenStreetMap)

**Purpose:** Road context enrichment

**Query Example:**

```
[out:json];
way(around:50,19.0760,72.8777)["highway"];
out geom;
```

**Returns:**

- Road name
- Road type (motorway, primary, secondary, etc.)
- Speed limit
- Surface type
- Lane count

**Rate Limit:** ~2 requests/second

---

### OSRM API (Routing)

**Purpose:** Worker navigation

**Endpoint:**

```
https://router.project-osrm.org/route/v1/driving/{lon1},{lat1};{lon2},{lat2}
```

**Returns:**

- Distance (meters)
- Duration (seconds)
- Encoded polyline
- Route geometry

**Note:** Coordinates are lon,lat (not lat,lon)!
